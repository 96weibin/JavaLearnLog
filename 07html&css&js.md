# Html,css,js

## html补充

* a标签

    ```html
        <a href="" target="" title="">link</a>
        <!--
            其中  target的属性值可以是   
            _self   在当前窗口打开，默认值
            _blank  在新窗口中打开
        -->
    ```

* 发送邮件

```html
    <a href="mailto:15764501228@163.com?subject=hello">发邮件给</a>
```

* map

    将img分为几个部分，点击跳转不同的链接，有点像canvas

* 表格的完整结构

```html
<table>
    <caption></caption>
    <thead></thead>
    <tfoot></tfoot>
    <tbody></tbody>
</table>
```
